Musik Romantis
================

Letakkan file MP3 Anda di folder ini.

Nama file yang direferensikan oleh aplikasi saat ini:

- `falling-in-love.mp3`

Anda dapat mengganti nama file tersebut, lalu ubah path di `script.js` pada fungsi `ensureMusic()`:

```js
bgMusic.src = 'assets/music/falling-in-love.mp3';
```

Jika menggunakan URL eksternal (CDN), cukup ubah `src` menjadi URL penuh.
